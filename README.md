# AI Evidence Quest

面向高中人工智能入门课的互动课程项目：学生用一个“校园资料问答助手”理解 RAG 的基本思想。

项目不依赖外部模型或 API。网页中的回答由本地资料块、检索分数和模板规则生成，目的是让学生看清楚：AI 问答不是凭空给答案，而是需要资料、证据、引用和不确定性边界。

在线演示：<https://sherkevin.github.io/ai-evidence-quest/>

## 课程主线

一个问题贯穿全课：

> 如果 AI 没有看过资料，它凭什么回答？

学生会依次完成：

1. 输入校园规则问题；
2. 观察系统如何检索资料；
3. 比较有证据回答和无证据回答；
4. 判断答案是否被资料支持；
5. 改写问题，让检索结果更准确。

## 快速打开

## 项目边界

这是课程原型和教学演示项目，不是正式上线的校园问答系统。示例资料为虚构校园资料，用于课堂活动。
