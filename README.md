# AI Evidence Quest：可信问答与资料检索项目课

这是一套面向高中人工智能课程、信息科技拓展课与科创社团的项目式教材。课程围绕一个完整作品展开：**让 AI 在回答校园资料问题时先检索证据、再组织回答，并在证据不足时明确拒答**。

## 课程主线

本书只解决一个核心问题：**AI 的回答怎样才能被检查、被追溯、被评价**。

学习者会完成一个“校园资料问答助手”。系统能够整理资料、切分资料、检索相关片段、生成带引用的回答，并记录回答质量。课程不把“会问大模型”作为终点，而把资料建设、检索策略、证据引用、拒答边界和评价量规作为主线。

## 教材结构

| 编 | 主题 | 核心产出 |
| --- | --- | --- |
| 第一编：可信问答系统 | RAG 系统地图、问题与资料、检索先于生成 | 系统结构图与资料卡片 |
| 第二编：证据检索实验 | 关键词、向量、混合检索、重排 | 可解释检索日志 |
| 第三编：回答与边界 | 引用、拒答、提示词、安全边界 | 带证据的回答模板 |
| 第四编：项目工程化 | 资料库、索引、评价、监控与缓存 | 可演示问答助手 |
| 第五编：科创拓展 | Agentic RAG、查询改写、评测报告 | 社团展示版作品 |

## 互动实验室

[证据检索实验室](https://sherkevin.github.io/ai-evidence-quest/labs/playground.html) 支持四类课堂操作：

- 编辑校园资料卡片，观察资料质量如何影响回答。
- 调节关键词、语义、来源可靠度和新鲜度权重。
- 比较 Top-K、拒答阈值、引用数量对结果的影响。
- 记录一次失败案例，并给出改进策略。
