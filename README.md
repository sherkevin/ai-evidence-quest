# AI Evidence Quest：给高中生的可信 AI 问答课

这是一套面向高中信息技术、人工智能校本课与科创社团的开源教材。它用一个完整项目贯穿全书。

**项目主线：**从零做一个“校园资料问答助手”，让 AI 先查资料、再回答，并在证据不足时承认不知道。

在线阅读与互动实验：<https://sherkevin.github.io/ai-evidence-quest/>

## 这本书解决什么问题

很多学生已经会向 AI 提问，但并不清楚 AI 的回答从哪里来、为什么会编造、怎样判断答案是否可信。本书不从模型参数或复杂公式讲起，而是把 RAG 拆成高中生可以观察和动手修改的流程：

1. 问题怎样影响检索；
2. 资料怎样被切成片段；
3. 相似度怎样决定“先看哪段资料”；
4. 回答为什么必须带引用；
5. 什么时候应该拒答或转交人工确认。

## 课程产出

学生完成后应能做出一个小型校园问答助手原型，并能解释：

- 一个回答是否有资料来源；
- 引用是否真的支持结论；
- 为什么“答得流畅”不等于“答得可信”；
- 如何设计低证据拒答规则；
- 如何用 badcase 改进资料、问题和回答规则。

## 教材结构
