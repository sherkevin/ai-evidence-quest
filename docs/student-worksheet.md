# 学习者任务单：人工智能为什么需要证据

## 任务 1：提出一个可检索问题

把一个模糊问题改写成资料中可能找到答案的问题。

- 模糊：我可以借东西吗？
- 更清楚：持校园卡一次最多可以借几本书、借多久？
