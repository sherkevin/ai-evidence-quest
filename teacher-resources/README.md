# 教师资源包

本目录保存教师备课和进阶代码阅读材料。它不是学习者第一遍阅读本书的主线，课堂展示建议优先使用根目录中的 `chapters/`、`docs/` 和 `labs/`。

## 目录说明

- `course-code/`：智能体、工具调用、检索增强生成、多智能体、安全与部署等代码示例和笔记本。
- `source-projects/`：用于备课和代码阅读的补充工程材料。
- `attribution/`：公开资源的原始说明和许可文本。
- `references.md`：教师备课延伸阅读。

使用这些材料时，建议先回到本书的中文章节，明确课堂目标和项目产出，再选择其中合适的代码片段作为进阶阅读。
